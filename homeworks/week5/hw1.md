## 前四週心得與解題心得

### 第一週
從接觸電腦以來幾乎都是使用有介面的軟體來操作，剛開始學習 CLI 很不習慣，打完指令後常常會覺得：「只要一行指令就完成了嗎？」、「真的有成功嗎？」，成功操作後都覺得很神奇，但又缺乏一點真實感

因為實在太害怕出錯，所以在第一次交作業時，我不知道反覆看了幾次影片才真的傳上去，結果馬上發生就錯誤，不斷出現 **The requested URL returned error: 403**，我翻遍文章、檢查 GitHub 設定、清除電腦紀錄，通通無解，最後終於按照 [這篇文章](https://docs.github.com/en/github/authenticating-to-github/creating-a-personal-access-token) 把得到的 token 當作密碼登入才成功交作業，第一次交作業就花了三、四個小時在找原因，差點嚇死


### 第二週
印象比較深刻的是我在二進位就卡住了，看很多次影片還是一直聽不懂，後來跑去問學過計算機概論的妹妹，她手把手的遠端教學，我才終於學會，只能說⋯有妹妹真好嗚嗚

這次接觸到 JavaScript，一次學到大量的函式，記都記不完，尤其是陣列函式，一直到學完第三週我還是聽不太懂，也慢慢感受到解不出題目的痛苦。有些題目甚至想破頭也解不出來，直到看解答才發現，解答的方法是完全沒學過的新方法，所以我開始洗腦自己看解答不是壞事，我看的其實不是解答，我看的是知識才對！

### 第三週
卡超久的一週，因為看到其他同學都是自己寫出來的，所以那時候我有著莫名的堅持，就是想全靠自己寫出來，可是在解 LIOJ 的過程很不順利，常常自我懷疑，偶爾會覺得：「都還沒真的開始寫程式就卡成這樣，我真的適合當工程師嗎？」

換個念頭想，一起學習課程的同學本來就不是站在同一個起跑點上，雖然我只學了一點 HTML/CSS，有的人卻已經學程式好幾年了，甚至也有人是完全沒接觸過程式的，轉念過後就不再懷疑自己了

這週學習到最猛的事情就是體驗到發散模式的威力，雖然 Huli 常常提醒大家解不出來可以去做其他事來放鬆心情，不過沒有感受過真的無法體會這句話的道理，直到解不出 LIOJ 時，我想著既然解不開那就睡吧，秉持著「沒有什麼事情是睡一覺不能解決的」來逃避解題，結果睡前玩手機到一半，忽然閃過靈感，馬上跳起來寫，寫法也完全和早上想的不一樣，最後竟然真的靠這個邏輯解出來！發散模式真的太猛啦。

### 第四週
依照前幾週寫筆記的經驗，花費的時間都太久了，想起第一週時 Huli 不斷提醒大家「這個課程是程式課程，並不是筆記課程。」，所以果斷放棄筆記，開始嘗試不寫筆記的成效。

在這週有好消息，也有壞消息。好消息是這週是四週以來我最快看完課程影片的一週，壞消息則是，第四週也是目前最挫折的一週，挫折的不是因為寫不出作業而難過，具體來說更偏向心靈層面上的挫折。

學習這週時剛好有參加到台中聚會，Huli 有建議可以先跳關到六、七週再回來，不過當時的我其實連第四週影片都還沒看完，第一直覺只認為自己應該不會跳關吧？！直到寫第二題作業時，呈現腦袋空白的狀態，真的連一行程式碼都寫不出來的那種空白，途中我甚至為了逃避學習而故意裝忙，突然開始大掃除、幫冰箱除霜等各種家事（明明平常根本不會特地做這些事啊啊啊），意識到自己這樣的心態不太對以後，決定打破無謂的堅持，趕緊先跳到第六週去，其實本來是不想跳週的，可惡耶⋯

直到完成第六週其中一個作業我才認命回歸第四週，不知道是不是有啟動發散模式的緣故，回頭重看第四週後，才頓時發現是自己當初根本看錯文件，找到原因也就順利完成 hw2 作業了，接著就抱著興高采烈的心情去看解答，此時又發現要寫出 error 的判斷式才對

```javascript=
if (err) {
 return console.log('抓取失敗', err);
}
```
印象中影片也沒提過，一度懷疑自己錯過了什麼嗎？！好奇之下問了兩個進度差不多的同學怎麼知道要寫？他們都回答：「下意識、直覺就加了」，原本我設想其他人應該都是跟我一樣，是看解答才發現的吧，結果收到這樣的回答震撼到我了，有種自己好不容易奮力跑到終點了，卻發現別人把跑步這件事當成喝水一般輕鬆，有點難以形容的挫折感 QQ

### HTTP Challenge
玩的過程覺得很像在解謎，很多小細節，連輸入錯誤的提示字也很有趣，但我先承認為了趕進度我只玩到第六、七題而已⋯希望自己未來有時間好好玩到破關

### 總結
當初報名時其實已經提完離職，本來是預估要全職學習的，結果被慰留後又心軟多待一段時間，在有工作的狀態下真的很難分配好時間，常常處在心有餘而力不足的狀況，心靈層面也很容易厭世，不過在第四週結束前，就按照原計劃離職了，果然還是想在這段時間內全心全意的學習，好好努力一次

學完四週後反而更期待後面的課程，很想趕快知道前四週學到的東西該怎麼應用在後面的課程，真的覺得加入課程的自己實在好幸運，不管是 Huli 的隨意聊、助教的精神喊話、同學們的進度報告，每一週都是滿滿的收穫
